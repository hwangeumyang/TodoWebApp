# Todo Web App

## 목적
Todo 리스트 관리
Web 기술 학습

## 기능
- Todo 생성
- Todo 리스트
- Todo 수정
- Todo 삭제
- 회원가입
- 로그인
- 로그아웃

## 키워드
- REST 백엔드, React 프론트엔드
- spring boot, gradle, maven repo, lombok, jpa
- JWT
- AWS 일래스틱 빈스톡, EC2, 오토 스케일링 그룹, 로드 밸런서, RDS
- Route 53, DNS 등록, DNS 로드 밸런서 연결
- OAuth 2.0 소셜 로그인

## 참고
React.js, 스프링 부트, AWS로 배우는 웹 개발 101 | 김다정

### 선정이유
배포에 쓰이는 기술 및 트렌드에 맞는 기술을 학습할 수 있음.
